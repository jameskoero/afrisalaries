# Security Policy

## Supported Versions
| Version | Supported |
|---|---|
| 1.x | ✅ Yes |
| < 1.0 | ❌ No |

## Reporting a Vulnerability
Do not open a public GitHub issue for security vulnerabilities.
Email jameskoero@gmail.com with a description, steps to reproduce, and potential impact.
You will receive a response within 72 hours.
